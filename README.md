# SANO BLD — Modern Heritage Portfolio

A refined, high-end web interface showcasing a collection of creative and technical projects. This portfolio, titled **"Modern Heritage"**, blends classic editorial aesthetics with modern interactive technologies.

## 🏛️ The Vision

Designed with a focus on typography and fluid motion, this hub acts as a bridge between minimalist design and functional performance. It features a sophisticated dual-theme system and an immersive user experience.

## 🚀 Projects Gallery

* **OpenTrad**: Open-source translation engine.
* **OmniMedia**: Comprehensive multimedia solution.
* **LastStats / Pulse / Aura**: A professional suite for music data and streaming.
* **SO'BÔHÈME**: High-end web showcase and branding.
* **BioLinkMaker**: Elegant link-in-bio utility for creators.
* **Eco-Drive**: Sustainable driving assistant.

## 🛠️ Technical Craftsmanship

* **Bespoke UI**: Custom-built interface using **Playfair Display** and **Montserrat** for a timeless look.
* **Adaptive Experience**: 
    * **Dual Theme Engine**: Seamlessly switches between "Light" and "Dark" with CSS variable injection.
    * **Command Palette**: Quick navigation and actions via a keyboard-driven interface (⌘ + K).
* **Performance Engineering**: 
    * Optimized asset loading.
    * Interactive Canvas particle system with a dedicated "Performance Mode".
* **Real-time Integration**:
    * **Firebase**: Live view tracking and analytics.
    * **Nager.Date API**: Context-aware date display and holiday recognition.
